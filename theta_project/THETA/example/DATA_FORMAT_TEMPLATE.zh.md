# THETA 数据格式模板

## 列名规范

| 用途 | 列名 | 适用模型 |
|------|------|---------|
| 文本 | `text` | 所有模型 |
| 时间戳 | `timestamp` | DTM |
| 协变量 | `cov_*` | STM |
| 标签 | `label` | 有监督 |

---

## DTM 模板

```csv
text,timestamp
"关于可再生能源政策的文档...",2020
"讨论人工智能发展的文章...",2021
"气候变化研究论文...",2022
"公共卫生政策文件...",2023
"数字经济趋势分析...",2024
```

---

## STM 模板

```csv
text,cov_province,cov_category,cov_source
"关于环境保护的政策文件...",北京,政策,政府
"讨论经济发展的新闻文章...",上海,新闻,媒体
"关于技术创新的研究报告...",广东,报告,研究院
"关于公共卫生的政府公告...",北京,政策,政府
"关于城市规划的学术论文...",江苏,研究,大学
```

---

## DTM + STM 组合模板

```csv
text,timestamp,cov_province,cov_category,label
"文档内容...",2023,北京,政策,环境
"文档内容...",2024,上海,新闻,科技
```
